# TP-final--UTN
Dashboard en Power BI para analizar la evolución y distribución de transferencias de vehículos en Argentina
📊 Análisis de Transferencias de Automotores en Argentina
📌 Descripción del proyecto
Este proyecto tiene como objetivo analizar la evolución de las transferencias de automotores en Argentina a lo largo del tiempo, utilizando Python para el procesamiento de datos y la generación de visualizaciones.
El análisis permite identificar tendencias, patrones estacionales y variaciones en el comportamiento del mercado automotor.
________________________________________
🎯 Objetivo del análisis
Analizar la evolución temporal de las transferencias de automotores para identificar:
•	Tendencias de crecimiento o caída
•	Patrones estacionales
•	Variaciones significativas en el tiempo
•	Distribución por tipo de vehículo y región
________________________________________
🧠 Hipótesis
•	Las transferencias presentan una tendencia variable en el tiempo, producto de decisiones y acontecimientos Macro y Micro económicos.
•	Existen patrones estacionales en determinados meses
•	El mercado presenta períodos con caídas pronunciadas
________________________________________
🗂️ Dataset
El dataset utilizado contiene información sobre transferencias de automotores en Argentina, incluyendo:
•	Año y mes de transferencia
•	Provincia
•	Cantidad de transferencias
________________________________________
🛠️ Herramientas utilizadas
•	Python
•	Pandas
•	Matplotlib
•	Seaborn
•	Google Colab
________________________________________
📊 Análisis realizados
Se llevaron a cabo los siguientes análisis:
📈 Evolución temporal
Se analizó la evolución de las transferencias a lo largo del tiempo para identificar tendencias.
📅 Análisis anual
Se evaluó el comportamiento de las transferencias por año.
🔍 Análisis mensual (2025)
Se estudió la variación interna dentro de un mismo año para detectar estacionalidad.
🌡️ Heatmap (año vs mes)
Se visualizaron patrones estacionales y variaciones interanuales.
________________________________________
📌 Principales conclusiones
•	El mercado automotor presenta variaciones a lo largo del tiempo, con períodos de crecimiento y caída.
•	Se identifican patrones estacionales, con meses de mayor y menor actividad.
•	Existen valores atípicos que afectan el comportamiento general de la serie.
•	La actividad se concentra en una determina provincia ( Buenos Aires).
•	El mercado es sensible a factores externos, lo que impacta en el volumen de transferencias.
